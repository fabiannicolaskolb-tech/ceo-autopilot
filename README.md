# Briefly – AI-Powered LinkedIn Personal Branding

Briefly is a SaaS platform that automates LinkedIn personal branding for executives. It combines AI-driven content ideation, scheduling, and analytics into a single workflow.

## Features

- **AI Ideation Lab** – Generate post ideas tailored to your expertise and audience
- **Post Library & Scheduler** – Draft, review, and schedule LinkedIn posts with approval workflows
- **Analytics Dashboard** – Track impressions, engagement, and growth with screenshot-based metric import
- **Voice Copilot** – Capture insights via voice and turn them into content
- **Creator Score** – Aggregated performance metric across reach, consistency, and engagement
- **Multilingual UI** – Full German/English language toggle

## Tech Stack

- **Frontend:** React · TypeScript · Vite · Tailwind CSS · shadcn/ui · Framer Motion
- **Backend:** Supabase (Auth, Database, Edge Functions, Realtime)
- **AI:** OpenAI integration via Edge Functions
- **Automation:** n8n webhook integrations

## Getting Started

```bash
npm install
npm run dev
```

Configure your Supabase project URL and anon key in `.env`.
